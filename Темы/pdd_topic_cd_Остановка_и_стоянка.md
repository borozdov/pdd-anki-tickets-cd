## Вопрос 2
**Разрешено ли Вам ставить автомобиль на стоянку в этом месте по нечетным числам месяца?**

![img](https://s.drom.ru/i24228/pdd/tickets/2016/1543885146.jpg)

1. Разрешено
2. Разрешено только после 21 часа
3. Запрещено

<details>
<summary>Показать ответ</summary>

**Правильный ответ:** 2. Разрешено только после 21 часа


<b>Правильный ответ: 2</b>
<br/>
            Действие происходит в населённом пункте, на дороге, имеющей по одной полосе движения в каждом направлении, без трамвайных путей посередине. На такой дороге можно осуществлять остановку и стоянку не только с правой стороны дороги, но и с левой, если нет запрещающих знаков. В данной же ситуации применены одновременно знаки: с правой стороны <a href="https://www.drom.ru/pdd/pdd/sign_3_29/" target="_blank">3.29</a> «Стоянка запрещена по нечётным числам месяца», с левой <a href="https://www.drom.ru/pdd/pdd/sign_3_30/" target="_blank">3.30</a> «Стоянка запрещена по четным числам месяца». При парковке автомобиля в таком случае следует учесть, что с 21.00 до 24.00 данные знаки не действуют из-за перестановки транспорта. Следовательно, в данном месте автомобиль можно поставить с 21.00 нечётного числа до 24.00 чётного числа.
Правильный ответ – второй.
Разрешено только после 21.00.
(«Дорожные знаки», пункт <a href="https://www.drom.ru/pdd/pdd/punkt_12_1/#p12_1" target="_blank">12.1</a> ПДД)        
</details>

## Вопрос 8
**В каком из указанных мест Вам разрешено пересечь сплошную линию разметки и остановиться?**

![img](https://s.drom.ru/i24228/pdd/tickets/2016/1543885406.jpg)

1. А и Б
2. Только Б
3. В указанных местах пересекать сплошную линию разметки запрещено

<details>
<summary>Показать ответ</summary>

**Правильный ответ:** 2. Только Б


<b>Правильный ответ: 2</b>
<br/>
            На данной дороге две смежные проезжие части. Между ними разделительная полоса. С левого края проезжей части нанесена разметка <a href="https://www.drom.ru/pdd/pdd/mark_1_2" target="_blank">1.2</a>, которая обозначает край проезжей части.

В месте «Б» - Вам можно останавливаться. Линия разметки <a href="https://www.drom.ru/pdd/pdd/mark_1_2" target="_blank">1.2</a> в данном случае обозначает край проезжей части. (Пункт <a href="https://www.drom.ru/pdd/pdd/punkt_1_2" target="_blank">1.2</a> термин «Разделительная полоса», «Горизонтальная разметка»)
