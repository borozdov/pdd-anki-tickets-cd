# pdd_topic_cd_Учебная_езда_Велосипеды

## Вопрос 1
**Разрешено ли обучать вождению на этой дороге?**

![img](https://s.drom.ru/i24228/pdd/tickets/2016/1543885118.jpg)

1. Запрещено
2. Разрешено, только при движении по крайней правой полосе проезжей части
3. Разрешено

<details>
<summary>Показать ответ</summary>

**Правильный ответ:** 3. Разрешено


<b>Правильный ответ: 3</b>
<br/>
            Движение происходит согласно знаку <a href="https://www.drom.ru/pdd/pdd/sign_5_1/" target="_blank">5.1</a> по автомагистрали. Учебная езда на ней не запрещена, т.е. разрешена.
(«Дорожные знаки», пункт <a href="https://www.drom.ru/pdd/pdd/punkt_16_1/#p16_1" target="_blank">16.1</a> ПДД).        
</details>

## Вопрос 2
**Какое оборудование должно иметь механическое транспортное средство, используемое для обучения вождению?**

1. Дополнительные педали привода сцепления (кроме транспортных средств с автоматической трансмиссией) и тормоза
2. Зеркало заднего вида для обучающего вождению
3. Опознавательные знаки «Учебное транспортное средство»
4. Все перечисленное оборудование

<details>
<summary>Показать ответ</summary>

**Правильный ответ:** 4. Все перечисленное оборудование


<b>Правильный ответ: 4</b>
<br/>
            Правильный ответ – все перечисленное оборудование, согласно Пункта <a href="https://www.drom.ru/pdd/pdd/punkt_21_5/#p21_5" target="_blank">21.5</a> ПДД и и Пункта 5 <a href="https://www.drom.ru/pdd/pdd/permission" target="_blank">Основных положений по допуску</a> 
</details>
