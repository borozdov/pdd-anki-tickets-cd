## Билет 4, Вопрос 4
**Вы буксируете неисправный автомобиль. По какой полосе Вам можно продолжить движение в населенном пункте?**

![img](https://s.drom.ru/i24228/pdd/tickets/2016/1543885574.jpg)

1. Только по правой
2. Только по левой
3. По любой

<details>
<summary>Показать ответ</summary>

**Правильный ответ:** 1. Только по правой

По ходу Вашего движения установлен знак <a href="https://www.drom.ru/pdd/pdd/sign_5_15_3/" target="_blank">5.15.3</a> «Начало полосы». По основной полосе знаком <a href="https://www.drom.ru/pdd/pdd/sign_4_6/" target="_blank">4.6</a> «Ограничение минимальной скорости» введено ограничение - минимальная скорость должна быть не менее 60 км/ч. Водитель, который не может продолжать движение с указанной скоростью, должен перестроиться на правую полосу. Буксирующим механическим ТС разрешается движение со скоростью не более 50 км/ч.
Вам следует продолжать движение только по правой полосе.
(«Дорожные знаки», пункт <a href="https://www.drom.ru/pdd/pdd/punkt_10_4/#p10_4" target="_blank">10.4</a> ПДД)
</details>

## Билет 5, Вопрос 4
**Нарушил ли водитель грузового автомобиля правила стоянки?**

![img](https://s.drom.ru/i24227/pdd/tickets/2016/1542608308.jpg)

1. Нарушил
2. Нарушил, если разрешенная максимальная масса автомобиля более 3,5&nbsp;т
3. Не нарушил

<details>
<summary>Показать ответ</summary>

**Правильный ответ:** 1. Нарушил

На околотротуарную стоянку, обозначенную знаками <a href="https://www.drom.ru/pdd/pdd/sign_6_4/" target="_blank">6.4</a> «Парковка (парковочное место)» и <a href="https://www.drom.ru/pdd/pdd/sign_8_6/#s8_6_4" target="_blank">8.6.4</a> «Способ постановки транспортного средства на стоянку» можно поставить указанным способом только легковой автомобиль или мотоцикл. Не путайте табл. <a href="https://www.drom.ru/pdd/pdd/sign_8_6/#s8_6_4" target="_blank">8.6.4</a> «Способ постановки ТС на стоянку» с табл. <a href="https://www.drom.ru/pdd/pdd/sign_8_4_1/#s8_4_3" target="_blank">8.4.3</a> «Вид транспортного средства», иначе допустите ошибку. 
(«Дорожные знаки»)
</details>

## Билет 38, Вопрос 4
**Вы можете продолжить движение по крайней левой полосе:**

## Билет 39, Вопрос 4
**Какие из указанных знаков разрешают выполнить поворот налево?**

![img](https://s.drom.ru/i24227/pdd/tickets/2016/1542609144.jpg)

1. Только А
2. А и Б
3. Б и В
4. Все

<details>
<summary>Показать ответ</summary>

**Правильный ответ:** 4. Все

Все знаки позволяют выполнить поворот налево. Знак «А» - <a href="https://www.drom.ru/pdd/pdd/sign_3_19/" target="_blank">3.19</a> «Разворот запрещен» - разрешает производить поворот налево. Знаки «Б» и «В» также не запрещают совершать маневр - поворот налево.
(«Дорожные знаки»)
</details>
