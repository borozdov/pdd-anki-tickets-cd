## Вопрос 1
**Сколько пересечений проезжих частей имеет этот перекресток?**

![img](https://s.drom.ru/i24227/pdd/tickets/2016/1542608752.jpg)

1. Одно
2. Два
3. Четыре

<details>
<summary>Показать ответ</summary>

**Правильный ответ:** 2. Два

Разделительная полоса примыкающей дороги справа делит данную дорогу на две проезжие части. Разметка <a href="https://www.drom.ru/pdd/pdd/mark_1_3/" target="_blank">1.3</a> (двойная сплошная линия) такого права не имеет.

## Вопрос 8
**По какой траектории Вам разрешено продолжить движение?**

![img](https://s.drom.ru/i24228/pdd/tickets/2016/1543885258.jpg)

1. Только по А
2. По А или Б
3. По любой из указанных

<details>
<summary>Показать ответ</summary>

## Вопрос 14
**Вы намерены проехать перекресток в прямом направлении. В данной ситуации:**

![img](https://s.drom.ru/i24228/pdd/tickets/2016/1543885251.jpg)

2. Вы имеете право проехать перекресток первым

## Вопрос 15
**Кому Вы обязаны уступить дорогу при повороте налево?**

![img](https://s.drom.ru/i24228/pdd/tickets/2016/1543885249.jpg)

1. Обоим транспортным средствам
2. Только автобусу
3. Только легковому автомобилю
4. Никому

<details>
<summary>Показать ответ</summary>

Перекрёсток неравнозначный. Главная дорога меняет направление. Транспортные средства, находящиеся на главной дороге, имеют преимущество, между собой руководствуются «правилом правой руки». У Вас помеха справа - уступаете автобусу.
(Пункты <a href="https://www.drom.ru/pdd/pdd/punkt_13_9/#p13_9" target="_blank">13.9</a>, <a href="https://www.drom.ru/pdd/pdd/punkt_13_10/#p13_10" target="_blank">13.10</a>, <a href="https://www.drom.ru/pdd/pdd/punkt_13_11/#p13_11" target="_blank">13.11</a> ПДД)

## Вопрос 16
**Разрешается ли водителю произвести остановку в этом месте в целях разгрузки транспортного средства?**

![img](https://s.drom.ru/i24228/pdd/tickets/2016/1543885248.jpg)

1. Разрешается
2. Разрешается, если остановка займет не более 5 минут
3. Разрешается, если не будут созданы помехи движению маршрутных транспортных средств
4. Запрещается

<details>
<summary>Показать ответ</summary>

**Правильный ответ:** 4. Запрещается

Полоса для маршрутных транспортных средств отделена прерывистой линией разметки. Другие случаи правилами не допускаются.
